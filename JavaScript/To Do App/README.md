# 📝 QuickStickz - Simple To-Do App

QuickStickz is a simple and interactive to-do app built using HTML, CSS, and JavaScript. It allows users to create, manage, and organize tasks using sticky notes.

---

## ✨ Features

- ➕ Add new sticky notes  
- ✏️ Edit tasks easily  
- 🗑️ Delete notes  
- 🖱️ Drag and move notes anywhere on the screen  
- 💾 Data stored in browser (local storage)  
- 📱 Responsive design  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript  

---

## 🚀 How to Use

1. Open the app in your browser  
2. Click on **Add Note**  
3. Write your task  
4. Drag notes to organize  
5. Delete when done
